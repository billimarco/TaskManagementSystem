After logging in, you will have the possibility to create your own personal repositories in which to create tasks. You will be able to see the list of repositories you belong to and interact with them. If that weren't enough, we did (or rather I did) ensure that every user could see the tasks assigned to him without having to enter every single repository. Ah... colors like this can seem not very good especially for the eyes. Unfortunately these are the bootstrap colors. (Spoiler: it could be done better only that I have to take other exams too... have mercy).

In each repository there will be the possibility to add new users who can interact with it (this also means they can be thrown out). Roles can also be managed, created and assigned to users who interact with the repository. Also remember that only if you have certain permits you can play certain functions in the repositories and that, since a user in a repository can be assigned multiple roles, the permissions accumulate. Obviously, needless to say, the tasks for each single repository can be managed, created, modified, etc. P.S. If you're wondering why a user can be forced into a repository by another user thus creating a vicious circle of possible spam ... well at the moment it's just like that, we'll think about it in a next version ... maybe.

There is not much to add compared to the repository part. In all cases, however, it will be possible to check the status of our tasks. Furthermore, the possibility of seeing the "history" of the task has been added so as to be able to trace its status over time. There doesn't seem to be much more to add. Having said that, have fun.
